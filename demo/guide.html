Manga Viewer 使い方ガイド

依存ゼロのWeb漫画ビューア。コピペ1ブロックで導入できます。本ガイドでは最小起動から全オプション、よくあるレシピまでを一気に把握できる構成にしています。

1. 5分で始める

以下を your-page.html に保存して、画像を3枚 (p1.jpg など) 同じディレクトリに置けば、ブラウザで開くだけで動きます。 

<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0, viewport-fit=cover">
</head>
<body>
  <div id="viewer"></div>

  <script type="module">
    import MangaViewer from 'https://cdn.jsdelivr.net/npm/@tokagemushi/manga-viewer@0.5.0/src/manga-viewer.js';

    new MangaViewer({
      container: '#viewer',
      pages: ['p1.jpg', 'p2.jpg', 'p3.jpg'],
      direction: 'rtl',           // 右綴じ（日本の漫画）
      title: 'サンプル漫画',
    });
  </script>
</body>
</html>
📝 メモ: CSS は viewer が Shadow DOM 内に自動展開するため、<link rel="stylesheet"> は不要です。

3. よくあるレシピ

A. 日本の漫画（右綴じ・見開き）

new MangaViewer({
  container: '#viewer',
  pages: [...],
  direction: 'rtl',
  title: '○○巻',
});

B. 西洋コミック（左綴じ・縦スクロール）

new MangaViewer({
  container: '#viewer',
  pages: [...],
  direction: 'ltr',
  viewMode: 'scroll',
});

C. スマホでも黒背景

theme: 'dark',

D. X共有・リンクコピーを消す

hideButtons: ['share', 'copy'],

E. 更新ボタンを追加

import MangaViewer, { icons } from '...';

new MangaViewer({
  // ...
  extraButtons: [{
    icon: icons.reload,
    label: '更新',
    onClick: () => location.reload(),
  }],
});

F. 戻るボタンを履歴で戻す

onBack: () => history.back(),

G. RTLで最終ページを右寄せに

direction: 'rtl',
lastPageAlign: 'start',  // start = 読み始め側 = RTLでは右

H. PWAでスライダーが iOS ホームバーに被る

v0.5.0 から自動16px加算。さらに調整したい場合:

#viewer { --mv-pwa-footer-bonus: 24px; }

I. 試し読み制限＋購入導線

previewLimit: 30,
purchaseUrl: 'https://your-shop.com/buy',
purchasePrice: '500円',

J. UI を英語化

messages: {
  resumeTitle: 'Resume reading?',
  resumeStart: 'Start over',
  resumeContinue: ' Resume',
  helpTitle: ' Help',
  helpSettings: '⚙ Settings',
  helpControls: '👆 Controls',
  bookmarkPanelTitle: 'Bookmarks',
  // ...必要なキーだけ上書き
},

K. CSS変数でカラー上書き

#viewer {
  --mv-bg: #1a1a1a;
  --mv-fg: #f0f0f0;
  --mv-accent: #ff6b6b;
}

利用可能な変数: --mv-bg, --mv-fg, --mv-text-muted, --mv-header-bg, --mv-footer-bg, --mv-btn-bg, --mv-btn-bg-hover, --mv-btn-fg, --mv-slider-track, --mv-spinner-track, --mv-spinner-fg, --mv-accent, --mv-shadow, --mv-footer-bottom-padding, --mv-pwa-footer-bonus

L. HTML/広告ページ挿入

pages: [
  'p1.jpg',
  { type: 'html', html: '<div>広告枠</div>', backgroundColor: '#fff' },
  'p2.jpg',
  { type: 'adsense', client: 'ca-pub-XXX', slot: '123456' },
],

HTML はホワイトリストサニタイザを通ります。スクリプトや危険なタグは除去されます。

4. トラブルシュート

症状原因対処
画面が真っ暗JS/CSSの読み込み失敗DevToolsコンソール / Networkタブで404を確認
アイコンが見切れるv0.4.0のサニタイザバグ（修正済）v0.5.0以上に更新
ヘルプ画面のフォントが違うv0.4.0で:hostにフォント未設定（修正済）v0.5.0以上に更新
しおりが保存されないプライベートブラウジング、localStorage容量超過通常モードで開く / bookmarkApiを実装
iOS PWAでスライダーが触りにくいホーム指示子と干渉--mv-pwa-footer-bonusを増やす
画像のロードが遅い大量URL同時読み込み画像CDN利用（loading="lazy"は内部で自動付与済）
